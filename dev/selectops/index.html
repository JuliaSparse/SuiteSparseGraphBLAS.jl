Select Operators

A SelectOp is effectively a binary or unary operation which is able to access the location of an element as well as its value. They define predicates for use with the select function.

Applying select with a SelectOp will always return a result with the same type and shape as its argument.

Built-Ins

Built-in SelectOps can be found in the SelectOps submodule.

SuiteSparseGraphBLAS.TRIL — Constant
select(SuiteSparseGraphBLAS.TRIL, A, k=0)
select(tril, A, k=0)

Select the entries on or below the kth diagonal of A.

See also: LinearAlgebra.tril
